# TKA Bahasa Indonesia Test Platform

## Overview
The TKA Bahasa Indonesia test platform is an online application designed to facilitate the assessment of students in Bahasa Indonesia, particularly for SD (Sekolah Dasar) and MI (Madrasah Ibtidaiyah) levels. It provides a comprehensive environment for both students and educators to evaluate proficiency in the language.

## Features
- **User-Friendly Dashboard**: An intuitive interface for users to navigate through different functionalities.
- **Customizable Tests**: Educators can create and modify tests to suit their educational goals.
- **Real-Time Scoring System**: Immediate scoring allows for quick feedback to students.
- **Progress Tracking**: Monitor student performance over time with detailed analytics.
- **Multi-Device Compatibility**: Accessible on various devices including tablets, laptops, and smartphones.

## Structure
1. **User Roles**: Differentiated access for students and educators.
   - *Students*: Access to take tests and view scores.
   - *Educators*: Create tests, view class performance, and provide feedback.
2. **Test Management**: Options for test creation, editing, and deletion by educators.
3. **Analytics Dashboard**: Visual representations of student performance metrics, including average scores, improvement over time, and more.

## Scoring System
- Each correct answer contributes to the overall score.
- Specific weight may be allocated for different sections of the test.
- Instant feedback on answers allows students to learn from mistakes immediately.

## How to Use the Dashboard
1. **Login**: Users need to enter their credentials to access the dashboard.
2. **Navigating the Dashboard**: Familiarize yourself with the tabs for test management, performance tracking, and settings.
3. **Taking a Test**: Students can select tests from the available list and complete them as per instructions.
4. **Viewing Results**: After completion, students can view scores and feedback in the results section.
5. **Creating a Test**: Educators can create a new test by selecting a 'Create Test' option, filling out the necessary details, and adding questions.

## Conclusion
The TKA Bahasa Indonesia test platform is a powerful tool for assessing language proficiency in an engaging way, promoting better learning outcomes for students.
